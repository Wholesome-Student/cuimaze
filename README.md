# cuimaze

## 対応端末
- 1920x1080以上を描画できるコンピュータ及びディスプレイ
- コマンドプロンプトを搭載している端末

## 導入方法
1. `g++`が導入されていない場合はインストール

2. 以下のコマンドでコンパイル
```bash
g++ -o cuimaze cuimaze_blank.cpp
```

3. ターミナルを全画面表示で開き、フォントサイズを9に設定

4. 実行
```bash
./cuimaze
```

## 操作方法
| キー | 操作 |
| :--: | :--- |
| W | 前に進む |
| S | 後ろに進む |
| A | 左に進む |
| D | 右に進む |
| J | 左を向く |
| K | 右を向く |
| M | マップの表示を切り替える |
| R | リトライ |
